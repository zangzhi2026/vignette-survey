# Vignette 实验问卷工具 - 规格说明 v2

## 概述

基于 PHP 的单文件问卷工具，专为 vignette 实验（情境实验）设计。支持：
- 随机分组（低/高自主组）
- 完整9页实验流程
- 内置所有量表题目
- 数据导出用于统计分析

## 实验流程（被试视角）

| 页 | 内容 | 说明 |
|---|---|---|
| P1 | 知情同意 + 指导语 | 固定文字 |
| P2 | 情境材料（随机显示低/高自主版本） | 管理员配置 |
| P3 | 操纵检验题（4题，5点量表） | 内置 |
| P4 | 主因变量：预期压力（4题） | 内置 |
| P5 | 中介：技术压力（4题） | 内置 |
| P6 | 中介：控制感丧失（3题） | 内置 |
| P7 | 情境真实性（1题）+ 注意力检验（1题） | 内置 |
| P8 | 人口学问题（4题） | 可配置 |
| P9 | 结束语 | 固定文字 |

## 内置量表

### 操纵检验（P3）
1. 情境中的 AI 系统能够自主规划任务流程。
2. 情境中的 AI 系统能自主做出判断，无需该员工频繁干预。
3. 情境中的 AI 系统在执行任务时拥有较高的决策权。
4. 在情境描述的工作流程中，主导权在 AI 一方而非员工。

### 预期压力（P4，主因变量）
1. 如果长期在这样的环境中工作，我会感到精疲力竭。
2. 如果长期在这样的环境中工作，一想到上班我就会感到疲惫。
3. 如果长期在这样的环境中工作，我会对工作的意义产生怀疑。
4. 如果长期在这样的环境中工作，我会对工作变得冷漠。

### 技术压力（P5，中介）
1. 在这样的环境中，我需要花很多精力去理解和验证 AI 的输出。
2. 在这样的环境中，跟上 AI 的运作方式会让我有压力。
3. 在这样的环境中，使用这套 AI 系统会让我感到紧张或焦虑。
4. 在这样的环境中，我会担心自己的技能变得过时。

### 控制感丧失（P6，中介）
1. 在这样的环境中，我对自己的工作几乎没有掌控感。
2. 在这样的环境中，重要决定不再由我做主。
3. 在这样的环境中，我只是在执行别人（AI）的安排。

### 真实性 + 注意力检验（P7）
- 刚才描述的工作情境真实可信。（5点量表）
- 这道题请直接选择"基本同意"。（注意力检验，答错剔除）

## 管理端功能

- 创建/编辑/删除实验问卷
- 配置：标题、描述、共同开头、低自主情境、高自主情境
- 设置人口学字段（年龄/性别/学历/AI使用经验等）
- 查看收集进度（每组多少份）
- 一键导出 CSV（带分组标记）

## 数据结构

### 问卷定义 (surveys.json)
```json
{
  "id": "exp_001",
  "title": "AI自主程度对员工倦怠的影响",
  "description": "情境实验研究",
  "common_intro": "请想象以下情境...",
  "scenario_low": "这套AI系统的角色是辅助您...",
  "scenario_high": "这套AI系统的角色是主导您的工作...",
  "demographics": ["gender", "age", "education", "ai_usage"],
  "created_at": "2026-05-31"
}
```

### 回答记录 (responses_xxx.json)
```json
{
  "participant_id": "P001",
  "group": "low",  // low | high
  "started_at": 1748640000000,
  "demographics": {"gender": "女", "age": "26-35", ...},
  "manipulation_check": [4, 5, 4, 5],  // 4题
  "anticipated_stress": [3, 4, 3, 4],  // 4题
  "tech_anxiety": [2, 3, 2, 3],         // 4题
  "loss_of_control": [4, 5, 4],        // 3题
  "realism": 4,                         // 1题
  "attention_check": 4,                // 1题（必须=4）
  "total_time": 360                    // 秒
}
```

## 技术选型

- 纯 PHP，无框架依赖
- JSON 文件存储
- 随机分组：服务端 `rand(0,1)`
- 数据导出：BOM + UTF-8 防止 Excel 乱码

## 部署

- 所有文件打包 zip，传至 PHP 空间
- data/ 目录需写权限
- 默认管理密码：admin123

## 验收标准

1. 管理后台能创建包含两种情境的实验问卷
2. 被试端自动随机分组，显示对应情境
3. 完成全部9页流程并记录答案
4. CSV 导出包含分组、所有量表、人口学、用时
5. 移动端界面正常显示
